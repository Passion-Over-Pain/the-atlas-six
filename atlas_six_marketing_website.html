The World Awaits

A Kingdom Divided.
A Party United.

The Atlas Six is a desktop fantasy RPG built without a game engine — forged in raw JavaScript, Electron.js, and friendship. It weaves branching narratives with tactical combat, cozy tavern evenings, and a murder mystery that will shake every throne in the realm.

"✦ A Note from the Creators" This game exists because six friends believed a story was worth telling. Every line of dialogue, every card in the battle deck, every flickering lantern in Bloomhaven — hand-crafted, late into too many nights.

The Heart of the Realm

Bloomhaven &
The Amber Flagon

Every great story begins in a tavern. Not every story ends there.

Bloomhaven is the quiet village at the edge of everything. A merchant crossroads, a refuge, and — as your party will discover — the epicentre of something deeply wrong.

The Amber Flagon tavern is your base of operations. Between battles, you'll rest here. Drink here. Argue about what to do next. Discover secrets in the floorboards and ghosts in the rafters.

On the third night in Bloomhaven, a body is found in the miller's cellar. No weapon. No forced entry. Six suspects — all of whom your party spoke to at dinner. The game changes here.

Technical Showcase

Built Without
a Game Engine.

The Atlas Six runs on raw JavaScript, Electron.js, and stubbornness. No Unity. No Godot. No shortcuts.

Architecture
Electron.js Desktop App
Full native desktop experience. Cross-platform from day one. Custom window chrome, native file I/O, and a rendering pipeline built from scratch.
Electron Node.js IPC
const win = createWindow({
  frame: false,
  titleBarStyle: 'hidden',
  webPreferences: { nodeIntegration: true }
});
Narrative
Recursive JSON Story Engine
A handbuilt dialogue and quest engine powered by nested JSON trees. Conditions, flags, branching paths, and consequence tracking — all in pure JavaScript.
JSON Recursion Event Flags
if (flags.has('bloomhaven.murder')) {
  return tree.branch('chapter_3b');
}
Persistence
Modular Save System
Chapter-aware save slots with delta compression. Snapshots preserve world state, relationship scores, inventory, and narrative flags across 100+ variables.
JSON.stringify fs.writeFile State Diff
Combat
Turn Engine & Card System
A fully custom initiative queue, AP system, and card-draw engine. Ability cards defined in JSON, rendered in HTML canvas, executed via a state machine.
State Machine Canvas API Queues
Events
Event Architecture
A global event bus connects narrative, combat, UI, and world systems without tight coupling. Every system is independently testable and swappable.
EventEmitter Pub/Sub Decoupled
bus.emit('world:npc_died', {
  id: 'fen', location: 'flagon'
});
Rendering
HTML/CSS Game UI
The entire interface — menus, inventory, battle screen, dialogue — is HTML and CSS. Pixel-perfect layouts, cinematic transitions, zero engine overhead.
HTML5 CSS Animations Vanilla JS
